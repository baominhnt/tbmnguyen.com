HCAHPS Healthcare Project
Quality-Survey Analysis

Analyzed national HCAHPS patient‑experience data to support hospital quality‑of‑care decisions. Built Spark pipelines to validate 5,000+ survey records, reconcile inconsistencies, and produce an analysis‑ready dataset. Used SQL to unify relational tables, resolve missingness, and generate segmented summaries across communication, responsiveness, and discharge‑planning domains. Conducted statistical and visual analyses in Python to identify correlation patterns and benchmark hospital performance against national standards. November, 2025

Heart Attack Predictions
Machine Learning Project

Developed a predictive framework in R to identify heart disease risk using the UCI Cleveland clinical dataset. Engineered a robust data pipeline to validate 303 patient records, resolve missing values, and factorize categorical variables for high-integrity modeling. Implemented and benchmarked multiple Supervised Machine Learning algorithms—including Decision Trees (rpart), Naive Bayes, K-Nearest Neighbor (KNN), and Random Forest—to evaluate diagnostic accuracy and model interpretability. Applied Unsupervised Learning through the Apriori algorithm to uncover non-linear association rules between symptoms and disease presence. Results demonstrated that the Decision Tree model provided the highest clinical reliability with 89.2% accuracy, significantly outperforming ensemble methods on this small-scale dataset.Delivered a final interpretive analysis that isolated Chest Pain Type (cp) and Major Vessel count (ca) as the primary drivers of heart disease, providing evidence-based risk thresholds to support early physician intervention and quality-of-care decisions.
